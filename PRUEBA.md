# Prueba del proyecto

## Contenido del repositorio
- Código fuente en C# / WPF.
- Librerías necesarias incluidas en Visual Studio.
- Archivos JSON de usuarios y configuración.
- Recursos gráficos y layout WPF.

## Instrucciones para usuario final

1. Ejecutar `CajeroPOC.exe` (generado desde Visual Studio).
2. Ingresar con un usuario existente:
   - Admin: Nombre `admin`, Contraseña `1234`
   - Cajero: Nombre `cajero`, Contraseña `1111`
3. Agregar productos al pedido y confirmar.
4. Observar la pantalla de pedidos en preparación y listos.
5. Generar ticket térmico y, opcionalmente, imprimirlo.
6. Probar gestión de usuarios (solo disponible para Admin).
7. Verificar que el reloj y el nombre de usuario se muestran correctamente.
